Build your name on the slot machine

Off-diagonal: a quantifier-style gamble (reels) used for language (letters). Three reels spin like a slot machine; when they pay out, one character is added. Each reel can miss independently (same odds); you only get a letter if all three hit. A spin is started from the keyboard, but which key is not shown—it changes after each spin. Enter only submits.
